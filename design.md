# MilkoSense - Design Style Guide

## Design Philosophy

### Visual Language
- **Scientific Precision**: Clean, minimal aesthetic reflecting laboratory accuracy and technological sophistication
- **Trust & Reliability**: Professional color palette and typography that instills confidence in scientific measurements
- **Modern Innovation**: Contemporary design elements that showcase cutting-edge AI and IoT technology
- **User-Centric Clarity**: Intuitive interface design prioritizing ease of use for both technical and non-technical users

### Color Palette
- **Primary Blue**: #2563EB (Scientific blue, trust, technology)
- **Secondary Blue**: #3B82F6 (Lighter blue for accents and highlights)
- **Neutral Gray**: #6B7280 (Professional text and secondary elements)
- **Light Gray**: #F3F4F6 (Background sections and cards)
- **White**: #FFFFFF (Clean backgrounds and contrast)
- **Success Green**: #10B981 (Quality indicators, positive results)
- **Warning Orange**: #F59E0B (Caution alerts, moderate quality)
- **Error Red**: #EF4444 (Critical alerts, poor quality)

### Typography
- **Display Font**: Inter (Modern, scientific precision for headings)
- **Body Font**: Inter (Clean, readable for all content)
- **Monospace**: JetBrains Mono (Technical data, sensor readings)

### Visual Hierarchy
- **Large Headings**: 48px, bold weight for main titles
- **Section Headings**: 32px, semibold for page sections
- **Subsection Headings**: 24px, medium weight
- **Body Text**: 16px, regular weight for readability
- **Technical Data**: 14px, monospace for sensor values
- **Captions**: 14px, regular weight for descriptions

## Visual Effects & Styling

### Core Libraries Integration
- **Anime.js**: Smooth micro-interactions and data transitions
- **ECharts.js**: Professional data visualization with scientific precision
- **p5.js**: Interactive sensor visualizations and particle effects
- **Pixi.js**: High-performance graphics for real-time dashboard
- **Typed.js**: Dynamic text effects for key messaging
- **Splide.js**: Smooth image carousels for educational content

### Animation & Effects
- **Micro-interactions**: Subtle hover effects on buttons and cards
- **Data Animations**: Smooth transitions for sensor readings and charts
- **Loading States**: Scientific-themed loading animations
- **Scroll Animations**: Gentle reveal effects for content sections
- **Particle Systems**: Subtle background effects representing data flow

### Header & Navigation Effects
- **Glass Morphism**: Semi-transparent navigation with backdrop blur
- **Active States**: Smooth color transitions for navigation items
- **Responsive Design**: Adaptive layout for all device sizes
- **Sticky Navigation**: Persistent access to key functions

### Interactive Elements
- **Sensor Cards**: 3D hover effects with depth shadows
- **Data Visualizations**: Interactive charts with hover tooltips
- **Form Elements**: Smooth focus states and validation feedback
- **Buttons**: Gradient backgrounds with hover transformations
- **Progress Indicators**: Animated progress bars and circular gauges

### Background & Layout
- **Consistent Background**: Clean white background throughout
- **Card-based Layout**: Elevated cards with subtle shadows
- **Grid System**: Responsive grid layout for content organization
- **Spacing**: Generous whitespace for scientific clarity
- **Borders**: Subtle borders and dividers for content separation

### Data Visualization Style
- **Chart Colors**: Muted blues and grays with high contrast
- **Grid Lines**: Subtle grid lines for precise data reading
- **Typography**: Clear, readable labels and legends
- **Interactive Elements**: Hover states and clickable data points
- **Animation**: Smooth data transitions and updates

### Mobile Responsiveness
- **Adaptive Typography**: Scalable text sizes for mobile devices
- **Touch-friendly Elements**: Appropriate button sizes and spacing
- **Simplified Navigation**: Collapsible menu for mobile
- **Optimized Charts**: Responsive data visualizations
- **Performance**: Optimized loading for mobile networks

- **Professional Trust**: Corporate color palette and typography
